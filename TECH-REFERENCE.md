# Build process.
1. Проверяем версию, чистим её (clean)
2. Нажмаем verify
3. Ждем, идем за кофе
4. Находим баг, плачем и сетуем на судьбу
5. Находим источник бага - правим
6. Загружаем изменения через build-run
7. Тестируем 
8. Рассказываем маме о своем успехе
